Program: Synchronous 4-Bit Counter with 3-State Outputs
Author: Jacob Simons
Date Published: November 2, 2022

This program represents the behavioral architecture of a Texas Instruments SN74ALS561A chip, which functions as a universal counter with 3-state outputs.
Originally produced as a laboratory assignment for ECE 4525 at Western Michigan University, designed for use with a Digilent Nexys A7 FPGA development board.
